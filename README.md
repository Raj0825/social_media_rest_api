# Social Media REST API (Spring Boot)

A basic **Social Media REST API** built using **Spring Boot** to demonstrate core backend concepts such as RESTful endpoints, HTTP methods, request/response handling, and in-memory data management.

This project is intentionally kept simple to focus on **REST fundamentals** before introducing databases and advanced features.
